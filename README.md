🚀 Overview

CareerSwipe is a web-based job recommendation system designed to simplify the job search process in Nepal. Instead of traditional searching, the system provides a swipe-based interface where users can explore and interact with job opportunities in an intuitive way.
The system uses content-based filtering techniques to recommend jobs based on user profiles, including skills, education, and preferences, making job discovery faster and more relevant.

🎯 Problem Statement

Traditional job portals require users to:
Manually search through large numbers of job postings
Spend time filtering irrelevant jobs
Deal with poor personalization

This leads to:

⏱️ Time-consuming job search
❌ Irrelevant recommendations
😓 Poor user experience

💡 Solution

CareerSwipe solves this by:
Providing personalized job recommendations
Introducing a swipe-based interaction system
Automatically matching users with relevant jobs

🎯 Objectives

Reduce mismatch between jobs and candidates
Minimize manual job searching effort
Provide a simple and interactive UI
Recommend jobs based on user preferences
Improve accessibility to job opportunities

⚡ How It Works
Users register and create profiles
Employers post job vacancies

System analyzes:
User skills
Job descriptions

Matching algorithm runs using:
TF-IDF
Cosine Similarity

Users swipe:
👉 Right = Interested
👈 Left = Skip

If matched:
Resume is sent to employer automatically
Notifications are generated

🧠 Algorithm Used

CareerSwipe uses Content-Based Filtering:
K-Means Clustering → Groups jobs & users
TF-IDF → Extracts important keywords
Cosine Similarity → Measures matching score
👉 This ensures accurate and relevant job recommendations
